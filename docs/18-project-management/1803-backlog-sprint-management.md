# BACKLOG & SPRINT MANAGEMENT

## Purpose

Определяет процессы управления Product Backlog, Sprint Backlog и выполнением задач в SHIK Platform.

---

# Objectives

- Transparent Prioritization
- Predictable Sprint Delivery
- Continuous Value Delivery
- Team Alignment
- Incremental Development

---

# Backlog Structure

Levels

- Initiative
- Epic
- Feature
- User Story
- Task
- Subtask

---

# Product Backlog

Contains

- New Features
- Improvements
- Bugs
- Technical Debt
- Security Tasks
- Infrastructure Tasks
- Documentation Tasks

---

# Sprint Backlog

Contains

- Sprint Goal
- Selected User Stories
- Development Tasks
- Testing Tasks
- Documentation Tasks

---

# Story Requirements

Every Story Must Include

- Business Value
- Description
- Acceptance Criteria
- Priority
- Estimate
- Dependencies

---

# Prioritization

Method

- Business Value
- Customer Impact
- Risk
- Technical Complexity
- Strategic Importance

Priority Levels

- Critical
- High
- Medium
- Low

---

# Sprint Planning

Activities

- Define Sprint Goal
- Select Stories
- Estimate Capacity
- Identify Risks
- Assign Tasks

---

# Sprint Execution

Daily Activities

- Daily Stand-up
- Development
- Testing
- Code Review
- Documentation Update
- CI Validation

---

# Sprint Review

Verify

- Completed Stories
- Acceptance Criteria
- Product Demonstration
- Stakeholder Feedback

---

# Sprint Retrospective

Discuss

- What Went Well
- What Can Improve
- Action Items
- Process Improvements

---

# Estimation

Methods

- Story Points
- Planning Poker
- Relative Estimation

---

# Work In Progress

Rules

- Limit Parallel Work
- Finish Before Starting New Work
- Minimize Context Switching

---

# Definition of Ready

Story Is Ready When

- Requirements Approved
- Acceptance Criteria Defined
- Dependencies Identified
- Estimate Completed

---

# Definition of Done

Story Is Done When

- Development Complete
- Tests Passed
- Documentation Updated
- Code Reviewed
- CI Passed
- Product Owner Accepted

---

# Metrics

Track

- Sprint Velocity
- Burndown
- Story Completion Rate
- Spillover Rate
- Lead Time
- Cycle Time
